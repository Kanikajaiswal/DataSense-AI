# DataSense AI 🚀

AI-Powered Business Analytics Dashboard built using React, Tailwind CSS, Node.js, Express, and Recharts.

---

## 📌 Features

- Drag & Drop CSV Upload
- Real-Time CSV Parsing
- Dynamic KPI Cards
- Search Functionality
- Category Filtering
- Interactive Charts
- Smart Business Insights
- Responsive Data Table
- Modern UI Design

---

## 📊 Charts Included

- Bar Chart
- Line Chart
- Pie Chart
- Area Chart

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Recharts
- Axios
- React Dropzone

### Backend
- Node.js
- Express.js
- Multer
- CSV Parser

## ⚙️ Installation

### Clone Repository

git clone https://github.com/YOUR_USERNAME/DataSense-AI.git

---

## 📦 Install Frontend Dependencies

cd client
npm install

---

## 📦 Install Backend Dependencies

cd server
npm install

---

## ▶️ Run Frontend

npm run dev

---

## ▶️ Run Backend

npm start

---

## 📈 How It Works

1. Upload CSV File
2. Dashboard parses CSV dynamically
3. KPIs generated automatically
4. Charts update in real-time
5. Search & filters work dynamically
